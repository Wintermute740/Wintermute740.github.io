Comments Enabled

November 27, 2024 — Wintermute

I don't like running a blog where I cannot interact with users, so comments are now enabled via Disqus. Unfortunately, it embeds 3rd-party JavaScript into the site in order to do so, but the only other way would be to do some forms, receiving those forms via email, and processing them via some hacked-together script. Since I prefer to moderate comments to keep spam away anyhow, this might not be so bad a solution, but Disqus is a little easier to do without much effort ;)
